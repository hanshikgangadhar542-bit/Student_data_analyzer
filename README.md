# Student Data Analyzer

This is a simple Python project that analyzes student data based on user input.

## Features
- Accepts student details (name, marks, etc.)
- Calculates results (average, total, etc.)
- Displays analysis output clearly

## How It Works
1. User enters student details
2. Program processes the data
3. Outputs results such as total and average
